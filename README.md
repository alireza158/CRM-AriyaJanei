
# 📊 سیستم مدیریت مشتریان و فروش - Laravel Edition (v1.0.0)

این پروژه یک سیستم CRM ساده و مدرن برای مدیریت مشتریان، محصولات، فاکتورها و گزارش‌کارهاست که با فریم‌ورک قدرتمند **Laravel** پیاده‌سازی شده و مناسب تیم‌های بازاریابی با نقش‌های مختلف است.

---

## 🛠️ تکنولوژی‌ها و پکیج‌های استفاده‌شده

- **Laravel 11** (فریم‌ورک اصلی)
- **Blade** (قالب‌سازی سمت سرور)
- **TailwindCSS** (رابط کاربری)
- **Laravel Breeze** (احراز هویت ساده و سریع)
- **Spatie Laravel Permission** (مدیریت نقش‌ها و سطوح دسترسی)
- **MySQL** (پایگاه‌داده)

---

## 👥 نقش‌های کاربری

- **Admin (مدیر):** دسترسی کامل به همه بخش‌ها
- **Marketer (بازاریاب):** فقط مشتریان خودش، فاکتور، گزارش‌کار
- **Guest (مهمان):** فقط امکان ثبت گزارش‌کار

---

## 📦 مدل‌های اصلی

- `Customer`: اطلاعات مشتری + دسته‌بندی + نحوه آشنایی
- `Product`: اطلاعات کالاها
- `Invoice`: فاکتورهای هر مشتری شامل محصولات و قیمت نهایی
- `Report`: گزارش‌کار کاربران
- `Category`: دسته‌بندی مشتریان
- `ReferenceType`: نحوه آشنایی با مشتری

---

## ⚙️ نصب و اجرای پروژه

### 1. کلون پروژه

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

### 2. نصب پکیج‌های PHP

```bash
composer install
```

### 3. نصب پکیج‌های فرانت‌اند

```bash
npm install
npm run dev
```

### 4. تنظیم فایل `.env`

```bash
cp .env.example .env
php artisan key:generate
```

و اطلاعات دیتابیس را در `.env` وارد کنید.

### 5. اجرای مهاجرت‌ها و seeder

```bash
php artisan migrate --seed
```

این دستور سه کاربر اولیه با نقش‌های مختلف می‌سازد.
